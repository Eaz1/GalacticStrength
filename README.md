StrengthSMP

A custom Spigot 1.12.2 plugin that adds a permanent Strength progression system.

Features

- Gain +0.5 hearts (+1 damage) per player kill.
- Lose 0.5 hearts (+1 damage) per PvP death.
- Maximum bonus of +5 hearts (+10 damage).
- Minimum bonus of 0.
- "/strength" command to view your bonus.
- "/strength top" leaderboard.
- "/strength resetall" for season resets.
- Saves player data across restarts.
- PlaceholderAPI support.
- Configurable messages and settings.
